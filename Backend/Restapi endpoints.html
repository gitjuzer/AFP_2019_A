/register - PUT

Leírás: regisztráljuk a felhasználót

Request body:

  • "user” : “felhasználónév”
  • “password” : “jelszó, kliens oldalon titkosítva”
  • “email” : “email cím,kötelező”

Return(JSON):

Sikeres regisztráció esetén:

  • “status” : “1”
  • “status message” : “User registered.”

Sikertelen regisztráció esetén:

  • “status : 0”
  •  “status_message”: “User failed to register.”
